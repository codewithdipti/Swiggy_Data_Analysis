#  Swiggy Data Analysis Dashboard 

##  Project Overview
The Swiggy Data Analysis Dashboard is a business analytics project developed to analyze food delivery data and extract actionable insights related to orders, revenue, customers, restaurants, and time-based trends.

This dashboard helps stakeholders understand customer ordering behavior, restaurant performance, and sales trends, enabling data-driven decision-making for operational and strategic improvements.

##  Business Problem
Online food delivery platforms like Swiggy handle large volumes of transactional data daily. However, without proper analysis, it becomes difficult to:

- Track overall business performance
- Identify top-performing restaurants and cuisines
- Understand peak order times and customer behavior
- Optimize delivery operations and marketing strategies
- Make data-driven decisions efficiently

### Problem Statement
To build an interactive analytics dashboard that consolidates Swiggy order data and provides insights into sales performance, customer behavior, restaurant efficiency, and time-based trends.

##  Dataset Description
The dataset represents a Swiggy-like food delivery system and includes:

| Dataset Component | Description |
|------------------|-------------|
| Orders | Order ID, order date, order value |
| Customers | Customer ID, location |
| Restaurants | Restaurant name, city, cuisine |
| Revenue | Order amount, delivery charges |
| Time | Order hour, day, month |

##  Key Performance Indicators (KPIs)
- Total Orders
- Total Revenue
- Average Order Value (AOV)
- Total Customers
- Total Restaurants
- City-wise Revenue
- Top Restaurants
- Popular Cuisines
- Peak Order Hours
